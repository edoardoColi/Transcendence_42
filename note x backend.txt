/* ============ Utenti ============ */
/login
Method post
Campi Json aspettati: email, password
Campi Json ritornati: key
Cosa fa: verifica se le credenziali esistono nel db e se si devi inviare una email con il codice: OTP e ritornare un codice JWT temporaneo
Codici ritornati:
    200 tutto ok
    204 credenziali sbaliate
    500 altri errori

/2af
Method post
Campi Json aspettati: codeotp
Campi Json ritornati: jwttoken
Cosa fa: verifica il codice 
Codici ritornati:
    200 tutto ok
    204 codice jwt sbagliato
    500 altri errori


/profilo
Method get
Campi Json aspettati: NULL (Nota: ti verra passato il JWT nella autorizzazioni)
Campi Json ritornati: email, nome, totalewin, totalelose, totalegame, {(array da max 20 elementi) nomep1, nomep2, esito, data, punteggio, gioco}
Cosa fa: ritorna le informazione del profilo: nota se non ci sono partite o in caso e erroe delle partite mettere: NULL
Codici ritornati:
    200 tutto ok
    500 altri errori

/modificaprofilo
Method post
Campi Json aspettati: nome, password
Campi Json ritornati: NULL
Cosa fa: se vuoto non modificare il campo
Codici ritornati:
    200 tutto ok
    204 nome gia in uso
    500 altri errori
